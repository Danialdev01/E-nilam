# E-nilam
E-Nilam adalah sistem perpustakaan yang memudahkan pihak admin perpustakaan Kolej Vokasional Kuala Selangor. E-Nilam yang merangkumi proses peminjaman buku, pengiraan statistik pelajar, pembacaan buku.
